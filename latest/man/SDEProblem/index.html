Defining a SDE Problem

To define an SDE Problem, you simply need to give the forcing function $f$, the noise function σ, and the initial condition $u₀$ which define an SDE

$$ du = f(u,t)dt + Σσᵢ(u,t)dWⁱ $$

f and σ should be specified as f(u,t) and σ(u,t) respectively, and u₀ should be an AbstractArray whose geometry matches the desired geometry of u. Note that we are not limited to numbers or vectors for u₀, one is allowed to provide u₀ as arbitrary matrices / higher dimension tensors as well. A vector of σs can also be defined to determine an SDE of higher Ito dimension.

Problem Type

# DifferentialEquations.SDEProblem — Type.

SDEProblem

Wraps the data which defines an SDE problem

$$ u = f(u,t)dt + Σσᵢ(u,t)dWⁱ $$

with initial condition $u₀$.

Constructors

SDEProblem(f,σ,u₀;analytic=nothing) : Defines the SDE with the specified functions and defines the solution if analytic is given.

Fields

  • f: The drift function in the SDE.
  • σ: The noise function in the SDE.
  • u₀: The initial condition.
  • analytic: A function which describes the solution.
  • knownanalytic: True if the solution is given.
  • numvars: The number of variables in the system
  • sizeu: The size of the initial condition (and thus u)
